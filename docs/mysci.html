My Open Science

It is important to practice what you preach! (See Open Science). In this section, I provide open access to sites which house my current/in-progress and completed research. The following resources provide an in-depth account of a project-much more than a manuscript or presentation might be able to.


ORCID

ORCID or Open Researcher and Contributor ID is an important tool used to access and track researchers across publications and disciplines. Check out more information about ORCID here.


GitHub

GitHub is an incredibly powerful tool for researchers, scientists and more across many fields. The role of GitHub can vary depending on the person. On my GitHub, you will find code used in a variety of projects. I primarily work with Python, MATLAB and R and the majority of my GitHub consists of these languages. GitHub is also where I house copies of my first-author presentations in a repository titled Presentations. Click on the icon above to access my GitHub.


OSF

The Open Science Framework (OSF) is a foundational tool in bolstering open science efforts. It is structured like a file folder system, with “Projects” being the main file, while subcomponents, like “Analysis” or “Code” being the files within. On my OSF, you will find a more detailed account of projects, an interface with GitHub to access code used in the project, and other components that typically make up a project, like the stimuli used and more. Click on the icon above to access my OSF.
